Tentang Krido

Halo! Perkenalkan saya, Krido

Halo saya Krido. Seorang yang menyebut dirinya sebagai Front-End Developer, padahal masih strugle juga, alias belum ‘ahli’ haha. Untuk saat ini saya sedang senang meng-eksplorasi tentang data science.

Saya menulis disini untuk diri saya sendiri, bukan untuk audiens saya. Disini saya berpikir untuk mengungkapkan hal-hal “tentang saya” dalam bentuk tulisan tentunya.

Jadi bisa dikatakan ini satu halaman besar yang mana isinya semua tentang diri saya. Dan suatu hari nanti blog ini akan menjadi semua yang tersisa dari saya, jadi sebaiknya saya menulis ‘autobiografi kecil’ saya sendiri.

Untuk sekarang saya tinggal di Banjarnegara(Jawa Tengah).
Saya(sangat) senang membaca, menulis, berbagi, ber-eksperimen(dan gagal) dengan hal-hal baru.

Untuk mengetahui apa yang sedang saya kerjakan di waktu luang saya, bisa lihat GitHub saya, dan untuk melihat tulisan saya, bisa lihat blog saya. Saya kadang menulis hal teknis di blog ini dan ini.

In real-life, jujur saya sebenarnya tidak se-ramah ini, lebih terkesan sombong dan cuek.
Jangan malu sungkan untuk menyapa saya khususnya melalui surat elektronik, karena saya senang berkomunikasi dan khususnya berkenalan dengan orang baru.

Seperti kebanyakan orang, saya juga mempunyai akun media sosial seperti Twitter dan Instagram

Terima kasih telah membaca halaman ini.
